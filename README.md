# PiFMX_VER
This repository is presented, all versions of [PiFMX](https://github.com/KOTYA8/PiFMX)

# Installation
Installation PiFMX:  
```bash
git clone https://github.com/KOTYA8/PiFMX_VER.git
cd PiFMX_VER/pifmxv(ver)/src
make clean
make
```
PiFMX launch:  
```
sudo ./pi_fm_x
```

# Version
* **V1** - Support **ECC**
